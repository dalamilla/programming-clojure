# Request Header Parser

Request Header Parser microservice.

## Instructions

To start this app:

- Build Application:

```
lein ring uberjar
```

- Running Application:

```
lein ring server-headless

```
or

```
javar -jar target/request-header-0.1.0-SNAPSHOT.jar
```

- Run Test Application:

```
lein test
```
